汇聚创意、分享热爱、连接同好

## 📖 项目简介

FURS 兽研会是一个面向兽迷（Furry）文化的交流平台网站，旨在为兽迷爱好者提供一个分享创意、交流文化、连接同好的线上空间。

**官方网站**: [https://furs-community.github.io/](https://furs-community.github.io/)

## ✨ 主要功能

- 🏠 **首页** - 展示最新兽聚信息和活动动态
- 🎉 **线下兽聚** - 发布和展示各地兽聚活动信息
- 📰 **最新新闻** - 更新兽圈相关新闻和资讯
- ℹ️ **关于 FURS** - 介绍兽研会的背景和理念
- 🎨 **FURS 作品** - 展示成员创作的优秀作品（建设中）
- 👥 **加入我们** - 提供加入社群的方式
